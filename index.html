ARTIST Website Options

Option 1 - Extremely Simple

This was more or less used to get most of the content into a web page. Some sections might benefit from the bullet points, where others will read better in paragraph form

Option 2 - More Design

Options 2 and 3 are very similar and incorporate more images, icons and pop-outs.

Option 3 - Variations to Option 2

Options 3 provides som different design elements but the flow is consistent with option 2.

Something to note / consider. Any of the styles from the options can be mixed; so, if you like the fonts from one, but the layout of another, we can merge them into one cohesive package. On the other hand, if you prefer other fonts (or style e.g. sarif/sans-sarif) you can select another font altogether.
